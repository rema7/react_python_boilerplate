### Quickstart

### Build server
##### Additional dependencies

Activate virtualenv and install python packages:

    $ mkvirtualenv virt-env -p python3
    (virt-env)$ pip install -r ./backend/.meta/packages
    (virt-env)$ cd ./backend
    

#### Run via docker container


    (virt-env)$ cd docker
    (virt-env)$ docker-compose up
